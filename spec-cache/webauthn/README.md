# WebAuthn Cache Index

## Normative Links

- W3C TR snapshot: <https://www.w3.org/TR/webauthn-3/>
- Editor's Draft: <https://w3c.github.io/webauthn/>

## Local Snapshot Files

The update script writes timestamped files to this directory:

- `webauthn-tr-<UTC_TIMESTAMP>.html`
- `webauthn-ed-<UTC_TIMESTAMP>.html`

It also updates rolling pointers:

- `webauthn-tr-latest.html`
- `webauthn-ed-latest.html`

## Notes

- Large files are expected in this directory.
- Keep snapshots for active work/review windows; prune stale snapshots during cleanup.
